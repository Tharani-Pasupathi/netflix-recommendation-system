# Netflix Recommendation System

## Project Overview
This project builds a Machine Learning-based recommendation system using the Netflix Prize dataset. It recommends movies to users based on historical ratings and collaborative filtering techniques.

## Objective
- Analyze Netflix user ratings.
- Build a movie recommendation system.
- Predict user preferences using Machine Learning.

## Dataset
- Netflix Prize Dataset
- Movie Titles Dataset

## Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Scikit-learn
- Surprise (SVD)

## Machine Learning Model
- Singular Value Decomposition (SVD)

## Project Workflow
1. Data Collection
2. Data Cleaning
3. Exploratory Data Analysis
4. Model Building
5. Recommendation Generation
6. Performance Evaluation

## Results
The recommendation system predicts user ratings and recommends similar movies based on learned user preferences.

## Future Improvements
- Deploy as a web application.
- Add content-based recommendations.
- Improve model performance using deep learning.
